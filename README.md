# Personal Finance Tracker

A desktop-based Personal Finance Tracker application developed using Java Swing and SQLite to help users manage income and expense transactions efficiently.

## Features

* Add and manage income and expense records
* Store transaction data using SQLite database
* Calculate total income, total expenses, and available balance
* Export transaction history to CSV files
* User-friendly graphical interface built with Java Swing

## Technologies Used

* Java
* Java Swing
* SQLite
* JDBC
* File Handling

## Project Structure

* Main.java
* FinanceTrackerGUI.java
* Transaction.java

## Future Enhancements

* Update and Delete transaction functionality
* Transaction search and filtering
* Monthly and yearly financial reports
* Charts and analytics dashboard
